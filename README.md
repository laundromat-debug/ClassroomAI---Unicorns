# ClassrmateAI---Unicorns

Classmate AI is an AI that helps you with your homework, helps you build schedules, and find clubs that interest you. This project is targeted towards newer studetns to help them immerse themselves in EPS fully. We have four pages in total, including a homepage, a clubs page where students can ask questions about clubs and find clubs they like, a homework page where students can use AI to understand their work, and finally a schedule builder where students can use AI to build a schedule with extracurriculars, classes and homework in mind.

How To Run:
1. Open the index.html page in live server
2. From the homepage, you can browse all the pages and find and use the page you need.
3. To use the clubs page, click on the photos of the clubs for a description and use the search bar to ask about clubs.
4. To use homework helper or clubs builder the instructions are on the page.

We used HTML, inline CSS, and some JavaScript as well as OpenAI API. We used AI in our program to help the student.
